NRL Pump

A probabilistic downgrader that buffers, reorders, and adds timing jitter to messages crossing security boundaries. Same topology as a data diode, but with active timing channel mitigation that defeats covert channels simple diodes leave open.

Security Properties

  • Batch accumulation — Messages are collected in groups of 4 before forwarding
  • Fisher-Yates reordering — Each batch is randomly shuffled to destroy ordering correlations
  • Pseudo-random jitter — Random delay added between batch releases to break timing channels
  • Unidirectional flow — Data flows HIGH to LOW only, no reverse path exists
Current Implementation

seL4 Microkit + C

The system description is identical in topology to a data diode. The pump logic — batching, reordering, and jitter — lives entirely within the nrl_pump protection domain's C implementation.

Key Code Highlight nrl_pump.c 
#define BATCH_SIZE 4

static msg_t batch[BATCH_SIZE];
static int batch_count = 0;

/* Fisher-Yates shuffle to destroy ordering correlations */
static void shuffle_batch(void) {
    for (int i = BATCH_SIZE - 1; i > 0; i--) {
        int j = prng_next() % (i + 1);
        msg_t tmp = batch[i];
        batch[i] = batch[j];
        batch[j] = tmp;
    }
}

void notified(microkit_channel ch) {
    if (ch == HIGH_CH) {
        batch[batch_count++] = read_msg(inbound_vaddr);
        if (batch_count == BATCH_SIZE) {
            shuffle_batch();
            add_jitter(prng_next() % MAX_JITTER);
            for (int i = 0; i < BATCH_SIZE; i++)
                write_msg(outbound_vaddr, batch[i]);
            microkit_notify(LOW_CH);
            batch_count = 0;
        }
    }
}

Fiducia Specification

The NRL Pump shares the same topology as the data diode in Fiducia — a simple HIGH to LOW one-way channel. The pump logic (batching, reordering, jitter) stays entirely within the C/Pancake implementation of the pump protection domain. Fiducia verifies the structural information flow guarantee.

  • Same topology as data diode — The .fis system declaration is structurally identical to the basic data diode
  • Internal pump logic in C/Pancake — Batching, Fisher-Yates shuffle, and jitter remain in the imported C implementation
  • Structural flow guarantee — Fiducia verifies at compile time that no reverse channel exists from LOW to HIGH
